# 기술 콘텐츠 품질 기준 (SotongElec)

- 문서 버전: 1.0
- 작성·확인일: 2026-07-22
- 적용 범위: 필기 강의·공식·문제·실기 학습자료

## 목적

콘텐츠 개수나 글자 수가 아니라 **학습자가 원리·계산·실무·안전을 이해할 수 있는지**를 기준으로 품질을 판단한다.

## 등급

| 등급 | 의미 |
|------|------|
| A | 교재·실무 참고 가능. 원리·공식·계산·사례·안전·확인문제가 주제에 맞게 갖춰짐 |
| B | 기본 학습 가능. 실무·진단·안전 보강 필요 |
| C | 요약·소개 수준. 구조는 있으나 깊이가 부족 |
| D | 템플릿·반복 문장 위주이거나 오류 가능성 |

글자 수만으로 A를 부여하지 않는다. 대표 10개 외를 자동 A 처리하지 않는다.

## A등급 최소 기준 (주제에 적합한 범위)

1. 내부 원리 설명 (물리적·회로적 의미)
2. 공식과 기호·단위·적용 조건
3. 단계별 계산 또는 회로/논리 해설 (주어짐→공식→단위→대입→결과→검산)
4. 실무 적용 사례
5. 잘못된 계산·결선·운용 사례
6. 진단·검산 또는 점검 순서
7. 안전 주의 (실제 위험 지점에 구체적으로)
8. 확인문제와 풀이 근거
9. 신뢰 가능한 출처·확인일 (법령·시험제도·KEC 관련)

불필요한 항목을 형식적으로 반복하지 않는다.

## 계산 설명 필수

- 주어진 값 / 구할 값 / 공식 / 단위 통일 / 대입 / 중간 / 최종·단위 / 검산 / 흔한 오답

가능하면 **두 가지 방법**으로 검산한다 (예: P=VI와 P=V²/R).

전선·차단기 선정은 단순 전류 계산과 별개임을 구분한다.

## 안전·법령

- 추측 수치 금지. 국가법령정보센터, KEC 공식 자료, 한국전기안전공사, KOSHA, Q-Net, 제조사 자료 우선
- 변경 가능 내용에는 확인일 기록
- 모든 문단에 동일 경고를 복붙하지 않음
